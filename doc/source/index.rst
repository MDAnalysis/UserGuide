Welcome to MDAnalysis User Guide's documentation!
=================================================

**MDAnalysis version:** |MDAnalysis_version|

**Last updated:** |today|

**MDAnalysis** (`www.mdanalysis.org`_) is an object-oriented python
toolkit to analyze molecular dynamics files and trajectories generated by
CHARMM_, Gromacs_, Amber_, NAMD_, LAMMPS_, `DL_POLY`_ and other
packages (see :ref:`formats` for full information). MDAnalysis can write
most of these formats, too, together with atom selections for use in
Gromacs_, CHARMM_, VMD_ and PyMol_ (see :ref:`selection-exporters`). It uses Numpy_ arrays to provide a flexible and relatively fast framework for complex analysis tasks.

.. _NumPy: http://numpy.scipy.org

.. _LAMMPS:  http://lammps.sandia.gov/

.. _PDB: http://www.rcsb.org/pdb/static.do?p=file_formats/pdb/index.html
