========================================
TRZ (IBIsCO and YASP trajectory)
========================================

.. include:: classes/TRZ.txt

MDAnalysis reads and writes `IBIsCO`_ / `YASP`_ TRZ binary trajectories in *little-endian* byte order.

.. _IBIsCO: http://www.theo.chemie.tu-darmstadt.de/ibisco/IBISCO.html
.. _YASP: http://www.theo.chemie.tu-darmstadt.de/group/services/yaspdoc/yaspdoc.html
